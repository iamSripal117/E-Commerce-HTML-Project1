🛒 E-Commerce Store (Pure HTML Project)

This project is a static multi-page E-Commerce website built using pure HTML only (no CSS, no JavaScript, no backend integration). The layout is structured entirely using HTML tables and basic tags to demonstrate fundamental HTML concepts.

📌 Project Overview

The website simulates a basic online shopping platform with:

Navigation header with logo and menu links

Product listing sections

About page

Contact page

Login page

Registration page

Footer with company details and policies

The design focuses strictly on HTML structure without external styling frameworks.

🚀 How to Run

Download or clone the repository.

Open home.html in your browser.

Navigate between pages using the header menu links.

🎯 Purpose of the Project

This project was created to strengthen foundational HTML skills by:

Building multi-page navigation

Structuring layouts without CSS

Understanding form handling structure

Practicing semantic organization using basic HTML elements
