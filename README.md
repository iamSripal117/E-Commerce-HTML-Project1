# RSR E-Commerce Static Website

## Overview

This project is a static e-commerce website built using pure HTML. It demonstrates the structure and layout of a basic online shopping platform, including product listings, user authentication pages, and informational sections.

The project focuses on front-end layout design using HTML tables and basic navigation without the use of CSS frameworks or JavaScript.

---

## Features

* Product listing page with multiple items and pricing
* Navigation bar with links to key sections
* User registration and login pages
* Contact information page
* About section with company and support details
* Footer with policies and company information
* Basic e-commerce UI elements such as cart and user icons

## How to Run

1. Download or clone the repository
2. Open `home.html` in any web browser
3. Navigate using the links provided in the header

No server or installation is required.

---

## Limitations

This project is a static implementation and does not include:

* Backend integration
* Database connectivity
* Authentication logic
* Dynamic cart functionality
* Responsive design optimization

---

## Purpose

This project is intended for:

* Learning HTML structure and layout design
* Demonstrating basic e-commerce UI concepts
* Academic or portfolio use
